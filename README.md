# e-com-merce
E-commerce site replicating eBay's UI and layout.

## Live demo:
https://e-com-erce.vercel.app/

## Tech stack:
* React.js, Next.js, Tailwind - Frontend 
* Supabase, Prisma - Backend
* stripe.js - simulation of payments
* Vercel - deployment
